# Benchmark_Club

**Benchmark_Club** — это веб-платформа, посвящённая сборкам компьютеров и подбору комплектующих. Пользователи могут создавать собственные конфигурации ПК, сравнивать железо, оценивать совместимость компонентов, а также изучать популярные сборки других участников.


##  Назначение сайта

Платформа создана для тех, кто собирает или подбирает системные блоки для различных задач: игр, стриминга, рабочих станций, бюджетных сборок или модернизации старых ПК.

Benchmark_Club помогает:

- Подбирать совместимые комплектующие для ПК.
- Создавать собственные сборки и сохранять их.
- Сравнивать процессоры, видеокарты, память, накопители и другое.
- Изучать рейтинги комплектующих и лучшие сборки других пользователей.
- Получать достижения и становиться частью сообщества ПК-энтузиастов.


##  Возможности

- Каталог комплектующих (CPU, GPU, RAM, PSU, материнские платы и др.)
- Создание и сохранение сборок
- Сравнение комплектующих
- Пользовательские профили
- Система достижений
- Геймификация прогресса

##  Установка (Windows)

> Перед началом убедитесь, что установлен Python версии 3.12 или новее.

1️ Клонируйте проект:
powershell
git clone <URL>
cd Benchmark_Club

2️ Создайте виртуальное окружение:
powershell
Копировать код
python -m venv venv

3️ Активируйте окружение:
powershell
Копировать код
venv\Scripts\activate

4️ Установите зависимости:
powershell
Копировать код
pip install -r requirements.txt

Запуск (Windows)
powershell
Копировать код

set FLASK_ENV=development
flask run
